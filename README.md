# Introducción a Python

Este repositorio contiene el material del curso **Introducción a Python**, utilizado como apoyo a los vídeos del canal de YouTube de Daniel Burrueco.

https://www.youtube.com/@DanielBurrueco

Aquí encontrarás **notebooks de Jupyter y ejemplos prácticos** que acompañan a cada uno de los vídeos del curso.

## Contenido del curso

El curso cubre, entre otros, los siguientes temas:

- Instalación de Python y Jupyter (Anaconda)
- Uso de Jupyter Notebook
- Variables
- Tipos de datos simples
- Tipos de datos estructurados
- Operadores
- Conversión de tipos
- Funciones integradas
- Estructuras de control (if, bucles)
- Introducción a librerías científicas

## Cómo usar este repositorio

No es necesario saber Git para usar este material.

Puedes:
- Descargar un notebook concreto y abrirlo en Jupyter
- Descargar el repositorio completo como archivo ZIP

Cada carpeta del repositorio corresponde a un bloque o vídeo del curso.
