Hasta ahora hemos escrito programas que ejecutan instrucciones de forma lineal. Pero en la práctica, casi cualquier programa necesita tomar decisiones para ejecutar cierto código solo cuando se cumple una condición. Un ejemplo sencillo sería comprobar si la temperatura de un motor supera un determinado umbral para activar una alerta si es necesario. Este tipo de lógica condicional aparece constantemente en programas reales.
