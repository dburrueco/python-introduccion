Las variables son uno de los pilares de cualquier lenguaje de programación, y en Python juegan un papel especialmente importante por su simplicidad y flexibilidad.

En este vídeo introducimos el concepto de variable desde una perspectiva intuitiva: qué es realmente una variable en Python, qué significa que una variable “apunte” a un objeto y por qué hablar de "almacenamiento" de un valor en una variable no es más que una metáfora útil. Todo ello con ejemplos sencillos y ejecutados directamente en Jupyter Notebook.

A lo largo del vídeo veremos, entre otros aspectos:

➤ Qué es una variable y cómo se crea en Python
➤ Diferencia entre el nombre de una variable y el objeto al que hace referencia
➤ Gestión automática de memoria en Python
➤ Uso de variables en expresiones y cálculos
➤ Tipado dinámico: una misma variable apuntando a valores de distinto tipo
➤ Cuándo es necesario usar print() y cuándo no en Jupyter
➤ Reglas de nomenclatura de variables
➤ Sensibilidad a mayúsculas y minúsculas
➤ Palabras reservadas del lenguaje
➤ Convenciones recomendadas por la guía de estilo PEP 8 (snake_case)
