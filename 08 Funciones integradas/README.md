Hasta ahora hemos trabajado con variables, tipos de datos, operadores y conversiones.
Pero, casi sin darnos cuenta, ya hemos estado utilizando algo fundamental en Python: funciones integradas.

En este vídeo explicamos qué son exactamente las *built-in functions*, dónde encontrarlas en la documentación oficial y por qué es tan importante conocerlas desde el principio.

▶️ Qué veremos en este vídeo

➡️ Qué son las funciones integradas y por qué siempre están disponibles
➡️ Diferencia entre funciones del núcleo de Python y funciones externas
➡️ Cómo consultar el listado oficial en la documentación
➡️ Por qué el listado puede variar según la versión de Python
➡️ Cómo leer (e interpretar) la documentación oficial

▶️ Ideas clave que aclaramos

* Las funciones integradas forman parte del núcleo del lenguaje.
* No necesitan instalación ni importación.
* Son independientes del dominio (no son específicas de ciencia de datos, desarrollo web, etc.).
* Algunas entradas del listado no son funciones en sentido estricto, sino clases (distinción que abordaremos en futuros vídeos).
* Es fundamental consultar siempre la documentación correspondiente a la versión de Python que tengamos instalada.

Conocer las funciones integradas significa dominar las herramientas básicas del lenguaje.
Casi cualquier programa en Python acabará utilizando varias de ellas. Además, aprender a navegar por la documentación oficial es una habilidad esencial para avanzar como programador.
