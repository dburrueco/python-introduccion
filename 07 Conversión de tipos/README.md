Python es un lenguaje de tipado dinámico, pero eso no significa que podamos mezclar tipos libremente en cualquier operación.

En este vídeo explicamos en profundidad cómo funciona la conversión de tipos, tanto la conversión implícita como la conversión explícita (type casting), y revisamos una a una las funciones principales que nos permiten transformar valores entre distintos tipos de datos.

Qué veremos en este vídeo:

➡️ Qué es la conversión implícita y cuándo la realiza Python automáticamente
➡️ Qué es la conversión explícita o casting
➡️ Diferencia entre coerción y conversión controlada
➡️ Uso de las funciones:

* `int()`
* `float()`
* `str()`
* `bool()`
* `list()`
* `tuple()`
* `set()`
* `dict()`

Aspectos importantes que analizamos:

➡️ Cómo `int()` trunca números reales (no redondea).
➡️ Qué ocurre al convertir textos que representan números.
➡️ Casos que generan errores y por qué.
➡️ Representación de valores especiales como "inf", "-inf" y "nan".
➡️ Cómo Python interpreta valores como verdaderos o falsos.
➡️ Qué ocurre al convertir estructuras como listas, tuplas, conjuntos o diccionarios.

Y revisaremos también una idea importante: Cuando aplicamos funciones como "list()" o "tuple()", no modificamos el objeto original, sino que creamos un nuevo objeto a partir de él. Comprender este matiz es esencial para evitar errores conceptuales más adelante.
