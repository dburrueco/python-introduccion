En este vídeo revisaremos una de las características más potentes y "pythonicas" del lenguaje: las comprehensions. Aprenderemos qué son y cómo nos permiten crear colecciones de valores a partir de iterables utilizando una sintaxis mucho más concisa y expresiva. Más concretamente:

⏩ Revisaremos cómo se suele crear una lista utilizando un bucle for clásico y el método append.List
⏩ Descubriremos la estructura básica para comprimir un bucle y una operación en una sola línea de código.
⏩ Veremos cómo añadir sentencias if para filtrar qué elementos del iterable queremos procesar, y cómo usar la estructura if-else para alterar el resultado generado.
⏩ Aprenderemos a aplicar esta misma lógica para construir Conjuntos (Set comprehensions) usando llaves y Diccionarios (Dictionary comprehensions).
⏩ Analizaremos el límite de esta sintaxis y descubriremos en qué punto una comprehension se vuelve un "jeroglífico ilegible", momento en el que es preferible volver al bucle tradicional.
