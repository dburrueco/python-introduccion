Jupyter Notebook es una de las herramientas más utilizadas en análisis de datos, ciencia de datos y aprendizaje automático. Su valor está en la posibilidad de combinar código, resultados y documentación en un mismo entorno, de forma interactiva y muy visual.

En este vídeo revisamos las herramientas esenciales de Jupyter, aprovechando al máximo la estructura basada en celdas.

➡️ Celdas: el núcleo de Jupyter
• Crear nuevas celdas antes o después de la celda seleccionada
• Ejecutar código paso a paso
• Diferencias entre modo edición y modo comando

➡️ Atajos de teclado imprescindibles
• Ejecutar una celda y avanzar
• Copiar, cortar y pegar celdas
• Mover celdas completas
• Deshacer y rehacer operaciones
Estos atajos aceleran el flujo de trabajo y convierten Jupyter en un entorno muy flexible.

➡️ Celdas de código y celdas Markdown
• Cómo alternar entre ambos tipos
• Crear títulos, listas y notas con Markdown
• Añadir explicaciones y estructura al cuaderno
• Integrar pequeños bloques de código formateado

➡️ Gestión del cuaderno
• Renombrar el archivo
• Guardar correctamente
• Cerrar Notebook sin dejar procesos abiertos
